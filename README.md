# Api-pokemon
Aplicación web desarrollada en JavaScript que consume la API de Pokémon utilizando Axios, Async/Await y Try/Catch para mostrar los primeros 20 Pokémon en tarjetas con imagen y nombre.

# Tarea N°7 - Consumo de API Pokémon

Aplicación web desarrollada con HTML, CSS y JavaScript.

## Funcionalidades

- Consumo de la PokeAPI.
- Visualización de los primeros 20 Pokémon.
- Cards con imagen y nombre.
- Uso de Async/Await.
- Uso de Axios para solicitudes HTTP.
- Manejo de errores mediante Try/Catch.

## API utilizada

https://pokeapi.co/api/v2/pokemon/

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Axios
